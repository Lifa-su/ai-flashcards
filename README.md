# 🧠 AI 闪卡 - 智能记忆工具

基于 AI 的闪卡/间隔重复记忆工具，纯前端单页应用。

## 功能

- **AI 生成闪卡** — 输入主题或粘贴笔记，DeepSeek AI 自动生成问答对
- **SM-2 间隔重复** — 经典 SuperMemo 算法自动安排复习时间
- **翻转动画** — 3D 卡片翻转效果
- **学习进度追踪** — 新卡片/学习中/已掌握/待复习状态
- **多卡组管理** — 创建、重命名、删除卡组
- **导入/导出** — 支持 JSON、CSV、Anki（制表符分隔）格式
- **键盘快捷键** — 空格翻转，1-4 评分，Esc 退出
- **响应式设计** — 适配桌面和移动端
- **本地存储** — 数据保存在浏览器 localStorage

## 使用

1. 直接用浏览器打开 `index.html`
2. 点击「设置」配置 DeepSeek API Key
3. 创建卡组 → AI 生成或手动添加闪卡 → 开始学习

## 技术栈

- 纯 HTML + CSS + JavaScript，无依赖
- DeepSeek API（`deepseek-chat` 模型）
- SM-2 间隔重复算法
- CSS 3D Transform 翻转动画
- localStorage 持久化

## 快捷键

| 按键 | 功能 |
|------|------|
| `Space` | 翻转卡片 |
| `1` | 重来 (Again) |
| `2` | 困难 (Hard) |
| `3` | 良好 (Good) |
| `4` | 简单 (Easy) |
| `Esc` | 退出学习 / 关闭弹窗 |

## 导入格式

**JSON:**
```json
[{"front": "问题", "back": "答案"}, ...]
```

**CSV:**
```
front,back
"什么是闭包?","闭包是..."
```

**Anki (制表符分隔):**
```
问题1\t答案1
问题2\t答案2
```
